# Cake Cafe

Cake Cafe is a modern cafe website built with Next.js and TypeScript. It is designed to present a bakery or dessert brand with a polished, responsive interface and a clear product-focused layout.

## Highlights

- Modern cafe and bakery website concept
- Responsive frontend for desktop and mobile visitors
- Product and brand presentation suitable for a real small business
- Live deployment for quick review and sharing

## Live Demo

https://cakecafe.vercel.app

## Tech Stack

- Next.js
- TypeScript
- React
- Modern CSS frontend styling
- Vercel

## Run Locally

```bash
npm install
npm run dev
```

Then open `http://localhost:3000` in your browser.

## Project Focus

This project demonstrates polished frontend work for a cafe or dessert business, with attention to presentation, responsiveness, and a clean user experience.
